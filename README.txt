Welcome to Decision Visualizer

To run Iris Visualizer

1. Go to Iris Visualizer Folder
2. Run a local server in folder (in python simply: python -m SimpleHTTPServer 8000) in this folder
3. View in Chrome at localhost:8000/iris_visual_scatter.html (May have to refresh once)

To run Generic Visualizer

1. Go to Generic Visualizer Folder
2. In command line write: 'python Generator.py X' where X is the filePath of your formatted csv
3. Run a local server (in python simply: python -m SimpleHTTPServer 8000) in this folder
4. To see your visualization, go to chrome and go to URL: localhost:8000
After visualization is done, exit browser and close local server. Delete the 2 JSON Files, and index.html.
